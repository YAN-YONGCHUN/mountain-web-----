## 🔍 浏览器控制台检查

### 打开开发者工具
按 `F12` 打开开发者工具

### Console 标签
**检查是否有错误**:
- ❌ 红色错误 = 有问题
- ✅ 无错误 = 正常

### Network 标签
**检查网络请求**:
1. 刷新页面
2. 查看 API 请求
3. 确认状态码:
   - ✅ 200 = 成功
   - ❌ 404 = 未找到
   - ❌ 500 = 服务器错误
   - ❌ CORS 错误 = 跨域问题

## 🐛 常见问题排查

### 问题 1: CORS 错误
**症状**: 浏览器控制台显示 CORS 错误

**检查**:
1. 访问测试页面测试 CORS
2. 确认 Netlify 已重新部署
3. 清除浏览器缓存

### 问题 2: 网络连接失败
**症状**: 显示"网络连接失败，请检查网络"

**检查**:
1. 确认 API 地址正确
2. 测试 API 是否可访问
3. 检查网络连接

### 问题 3: 登录失败
**症状**: 显示"账号或密码错误"

**检查**:
1. 确认使用正确的测试账号
2. 手机号: `13800138000`
3. 密码: `123456`

### 问题 4: 数据不显示
**症状**: 页面加载但无数据

**检查**:
1. 打开浏览器控制台
2. 查看 Network 标签
3. 确认 API 返回数据
